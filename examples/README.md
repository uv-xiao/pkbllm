# Examples

This folder holds example materials generated by pkbllm skills.

## `examples/materials/` (git submodule)

This is an example `$HUMAN_MATERIAL_PATH` repo containing:

- `research/{flashinfer,nanoflow,sglang}/repo_analysis.md`
- `research/{flashinfer,nanoflow,sglang}/hands_on/mlkb-repro/…`
- `exercises/tutorials/{flashinfer,nanoflow,sglang}/…`

### Note on the submodule URL

On this machine, the submodule URL is configured as a local path (see `.gitmodules`).
If you want this to be cloneable elsewhere, point it at a remote repo you control and
update the submodule reference.
